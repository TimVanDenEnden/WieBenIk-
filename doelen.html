Mijn doelen!

Een van mijn doelen is om zo goed mogelijk te worden in progameren van websites. Mijn tweede doel dat ik graag op korte termijn wil berijken is om mijn auto rijbewijs te halen! Mijn derde doel is het halen van mijn mbo goed resultaten af te maken. Mijn vrijtijds doel is nog beter te worden in gamen.

Wat ik nog meer van wil leren!

HTML 5:

Ik wil deze vaardigheid nog beter leren. Zodat ik het foutloos kan gebruiken. Want het is een belangrijke basis om de rest van website bouwen. En om te kunnen begrijpen. Je kunt er makkelijk een website mee bouwen die niet zo veel hoeft te kunnen. En het is niet zo heel moeilijk. En hier door vind ik het super leuk!

CSS 3:

Elke website moet er goed uitzien. Daarom zou ik het graag nog beter en er in veder willen leren. Om het nog beter te kunnen begrijpen en te kunnen gebruiken. Alles wat je ziet aan opmaak is CSS. En je kunt er heel veel mee. Hier zijn een paar voorbeelden: letter type aanpassen andere kleurtjes enzovoort. Maar je kan er ook animatie mee toevoegen en afbeelding mee toevoegen. Je kan er nog veel meer mee en dat maakt het vind ik heel erg leuk.

PHP:

Met PHP kun je geweldige functions maken die je op je website kan gebruiken. PHP is niet hoe dingen er uit zien maar gaat er vooral om hoe dingen werken. Dingen die gebeuren als je ergens op klikt bijvoorbeeld. Een paar voorbeelden zijn: inlog systeem en forums en nog veel meer. Daarom zou ik hier door ook graag nog veel meer van willen leren.

MYSQL:

Bij PHP werk je meestal met gegevens en die moet ergens vandaan halen. Dat kan je doen door middel van MySql. Een Sql is een stukje informatie die je bijvoorbeeld wilt bewaren. Die stuur je dan naar een database en dan wordt het bewaard. Maar het voordeel er van is om het in de database te bewaren is. Dat je het overal weer kan terug kan krijgen. Ik vind dit super cool hoe het werkt en daarom zou ik er graag in veder willen leren.

JavaScript:

Met javascript kan je stukjes software gebruiken op de pc waarmee je de website bezoekt. Bij voorbeeld een klokje op je website die echt met de secondes laat zien enzo. Je kan er nog veel meer mee het is aardig uitgebreid. Ik zou hier graag meer van willen leren. Omdat je website nog nuttiger mee kan maken.

AJAX:

AJAX is heel handig te gebruiken om je te update zonder te reloaden Een voorbeeld is bij een zoek functie de bij elke leter die je intype nieuw zoek resultaten weer geeft. Ik heb hier persoonlijk nog heel weining mee gewerkt. Maar ik zou hier heel graag mee veder willen leren.

PhotoShop:

Soms heb je wat foto's nodig op je website. En meestal zijn ze of nog niet helemaal goed of je wilt er iets aan aanpassen. Dat kan je PhotoShop zo wat alles er mee aanpassen. Het is een heel uitgebreid programma. En het vereist aardig wat kennis om er goed mee te kunnen werken. Daarom lijkt het mij een leuk iets om te leren.

Search Engine Optimization:

Als je een leuk website heb gebouwd moet hij natuurlijk ook goed te vinden zijn. dat kan je doen doormiddel van zoektermen te gebruiken tags te gebruiken. Persoonlijk heb ik hier heel weinig mee gewerkt. Maar vind dit wel een belangrijk iets en zo het daarom graag willen leren.
